Este repositório armazena os materiais visuais suplementares (animações e GIFs) do Trabalho de Conclusão de Curso (TCC) intitulado **"Desenvolvimento de Jogos Eletrônicos: Uma Abordagem Baseada em Modelagem Formal"**.

## Sobre o Projeto (O TCC)

Este repositório serve como um anexo visual para o TCC do curso de **Ciência da Computação** da Universidade Federal de Jataí (UFJ).

O objetivo do trabalho principal é propor e validar uma metodologia de modelagem e análise de cenários de jogos utilizando **Redes de Petri Coloridas Hierárquicas (CPNs)**. As animações aqui presentes demonstram a execução da simulação passo a passo do modelo desenvolvido, validando o fluxo de progressão do jogador, o uso de itens e a corretude estrutural do modelo. A gameplay do jogo Dark Souls 1 foi retirada do video "Dark Souls - Longplay Full Game 100% Walkthrough [No Commentary] 4k" no link: https://www.youtube.com/watch?v=WyD9BeFFTw0

## Conteúdo do Repositório

Este repositório não contém código-fonte para compilação, mas sim os arquivos de mídia utilizados como figuras no documento do TCC. O conteúdo inclui:

* **Vídeos de Referência (Gameplay):** Clipes curtos do jogo (*Dark Souls I*) que mostram as ações reais correspondentes a cada etapa da simulação no modelo.

## Tecnologias Utilizadas (Para o TCC)

* **Redes de Petri Coloridas (CPNs):** Para a modelagem formal da lógica do jogo.
* **CPN Tools:** Ferramenta utilizada para a modelagem, simulação e análise de *State Space* do modelo.
* **LaTeX:** Para a escrita do documento acadêmico.
